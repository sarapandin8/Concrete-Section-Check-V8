# UI.COMMERCIAL.TABS2 — Dark-blue bold typography for existing tabs, buttons, and input labels

## Scope
Visual-only CSS hardening across the existing Streamlit app shell.

## Included
- Existing app/workspace tabs remain in the same positions and keep the same navigation logic.
- Existing tab/radio labels are bolder, slightly larger, and use dark-blue text.
- Existing action buttons and download buttons use bold dark-blue text.
- Input-focused labels for number inputs, text inputs, select boxes, checkboxes, toggles, uploaders, and editable tables use dark-blue bold text.
- Field display text in Streamlit/BaseWeb inputs is dark-blue and slightly heavier.

## Out of scope
- No new tabs.
- No navigation rearrangement.
- No solver, geometry, loads, analysis, report, project schema, rebar, or prestress changes.
