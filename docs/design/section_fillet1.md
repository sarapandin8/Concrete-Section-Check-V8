# SECTION.FILLET1 — Rectangular Filleted basic solid preset

## Scope
Add a new `Rectangular Filleted` section preset under `Basic Solid` for Column/Pier/Wall/Pylon workflows.

## Included
- New geometry generator `rectangular_filleted`
- New dimension helper `rectangular_filleted_dimensions`
- Preset wiring in `data/section_presets.json`
- Regression tests for geometry validity, area, and dimension symbols

## Behavior
- Overall width `B`
- Overall height `H`
- Uniform corner fillet radius `R` at all four outer corners
- Polygon generated by rounded-rectangle helper with analytical dimension guides `B`, `H`, and `R`

## Out of scope
- No solver changes
- No PMM/shear/torsion/SLS/report changes
- No inner void / hollow / mixed-radius support in this milestone
