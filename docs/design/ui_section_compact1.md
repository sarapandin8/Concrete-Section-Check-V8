# UI.SECTION.COMPACT1 — Section Builder compact working layout

## Scope
This milestone polishes the Section Builder working layout only.

## Changes
- Stack geometry parameters and gross section properties in the left column.
- Keep the live section preview in the right column.
- Reduce Section Builder preview height from the default visualization height for this page only.
- Replace the long preview validation/status panel with compact status cards.
- Preserve detailed preview validation in an expander when errors/warnings/info exist.
- Make section property cards auto-fit with smaller card padding for denser working-screen use.

## Out of scope
- No geometry generator changes.
- No section-property formula changes.
- No PMM, shear, torsion, SLS, report, rebar, prestress, loads, or project schema changes.
