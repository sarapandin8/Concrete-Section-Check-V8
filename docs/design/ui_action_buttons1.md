# UI.ACTION.BUTTONS1 — Highlight primary action buttons

## Scope
Improve visual hierarchy for action/confirmation controls without changing calculation behavior.

## Included
- Soft highlighted primary button style for Run, Save, Load/Upload, Apply, Replace, Append, Add, Update, and other confirm-style actions.
- Primary action text is bold and dark blue.
- Upload file browse buttons receive the same soft action styling.
- Key actions are marked as primary where appropriate: PMM Run, Save Project, Load Project JSON, and project info update.

## Out of scope
- No navigation behavior changes.
- No solver, geometry, load, report, rebar, or prestress calculation changes.
- No broad workflow refactor.
