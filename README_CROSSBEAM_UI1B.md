# CROSSBEAM.UI1B — Full-length dashed hollow-void elevation

This scoped visual milestone updates only the Portal Frame Crossbeam Segment Layout elevation figure.

## Change

- Hollow-segment void boundaries now extend from the segment start station to the segment end station.
- Hidden void boundaries are shown with dashed lines.
- The former inset white solid-outline cut-out is removed.
- Hollow segments retain their lighter fill for quick Solid/Hollow recognition.

## Scope guard

No section geometry, section properties, segment assignment data, tendon geometry, solver routing, Project JSON behavior, or existing non-Crossbeam workflow is changed.
